# Game Files Not Included

This release does not contain or link to copies of NBA 2K16 game content, including:

- `NBA2K16.exe`, original game DLLs, Steam files, DRM-bypass or `NoDVD` files.
- Extensionless game archive/container files such as `0A`, `0B`, `1P`, and similar files from the game folder.
- Extracted assets, original roster saves, or files copied from another user's game installation.

The viewer works with a user's own local installation.
